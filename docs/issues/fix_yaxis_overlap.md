# SGE图表三条线重叠问题分析

**问题时间**: 2026-03-28 23:21  
**问题描述**: 用户反馈SGE溢价走势图中三条线看起来完全一样

---

## 问题原因

### 1. 数据完全没有变化

**检查结果**：
```
最近10条数据（晚上23:21）：
- 人民币金价: 1002.7116 元/克 (相同)
- 国际金折算: 998.8127 元/克 (相同)
- 溢价: 3.8989 元/克 (相同)

波动幅度: 0.0000 元/克
```

**原因**：市场休市，价格冻结

### 2. Y轴配置问题

**左Y轴（价格）**：
- 范围：800 ~ 1200 元/克（固定范围）
- 跨度：400 元/克

**数据差异**：
- 人民币金价：1002.71 元/克
- 国际金折算：998.81 元/克
- **差值：3.9 元/克**

**视觉效果**：
```
差值占Y轴比例 = 3.9 / 400 = 0.975%

在400元/克的刻度范围内，3.9元的差异只占不到1%，
肉眼几乎无法分辨！
```

### 3. 图表设计问题

**当前设计**：
- 左Y轴：人民币金价 + 国际金折算（两条线重叠）
- 右Y轴：溢价（单独一条线）

**问题**：
- 左Y轴的两条线（绿色和橙黄色）差值只有3.9元
- 在固定的800-1200范围内，几乎完全重叠
- 用户看到的就是"三条线一样"

---

## 解决方案

### 方案A：优化Y轴范围（推荐）

**当前配置**：
```javascript
yAxis: [
  {
    type: "value",
    min: 800,           // 固定最小值
    max: 1200,          // 固定最大值
    // ...
  }
]
```

**优化后**：
```javascript
yAxis: [
  {
    type: "value",
    scale: true,        // 自动缩放
    min: function(value) {
      return value.min - 10;  // 最小值-10
    },
    max: function(value) {
      return value.max + 10;  // 最大值+10
    },
    // ...
  }
]
```

**效果**：
- 修改前：Y轴 800-1200（400跨度）
- 修改后：Y轴 990-1010（20跨度）
- **差异放大 20倍！**

**当前时间**：23:21（夜盘时段）

**问题**：虽然在夜盘时段，但价格未更新，可能：
1. 新浪API数据延迟
2. 实际市场波动很小
3. 数据源问题

**建议**：等到明天日盘（9点后）再观察

### 方案C：调整数据采集策略

## 立即修复：优化Y轴

让我实施方案A，优化左Y轴的范围，使差异更明显。

### 修改内容

**文件**：`static/app.js`

**位置**：`renderSgeChart()` 函数

**修改**：
```javascript
// 修改前
yAxis: [
  {
    type: "value",
    min: yMin,  // 800
    max: yMax,  // 1200
    // ...
  }
]

// 修改后
yAxis: [
  {
    type: "value",
    scale: true,
    min: function(value) {
      return Math.floor(value.min - 10);
    },
    max: function(value) {
      return Math.ceil(value.max + 10);
    },
    // ...
  }
]
```

**预期效果**：
- Y轴自动适配数据范围
- 放大细微差异
- 三条线不再重叠
