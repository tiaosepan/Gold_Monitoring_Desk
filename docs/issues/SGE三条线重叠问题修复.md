# SGE 三条线重叠问题修复

**修复时间**: 2026-03-28 23:21  
**问题**: SGE溢价走势图中三条线看起来完全一样

---

## 问题分析

### 数据检查结果

```
当前时间: 23:21 (深夜，市场休市)

最近10条数据：
人民币金价: 1002.7116 元/克 (完全相同)
国际金折算:  998.8127 元/克 (完全相同)
溢价:          3.8989 元/克 (完全相同)

波动幅度: 0.0000 元/克 (完全没有变化)
```

### 根本原因

#### 1. 数据确实不同

三条线的数值**实际上是不同的**：
- 人民币金价：1002.71 元/克
- 国际金折算：998.81 元/克
- **差值：3.9 元/克**

#### 2. Y轴范围过大

**修改前**：
- Y轴范围：800 ~ 1200 元/克
- 跨度：**400 元/克**

**问题**：
```
差值占比 = 3.9 / 400 = 0.975%

在400元/克的刻度范围内，3.9元的差异只占不到1%，
肉眼根本无法分辨！
```

#### 3. 市场休市

- 当前时间：23:21
- 沪金市场：已休市
- 数据冻结：价格不再更新
- 所有采集的数据都是相同值

## 修复方案

### 核心修改：Y轴自动适配

**修改文件**: `static/app.js`

**修改前**：
```javascript
// 固定Y轴范围
const yMin = state.sgeYAxisRange?.min ?? 800;
const yMax = state.sgeYAxisRange?.max ?? 1200;
```

**修改后**：
```javascript
// 自动计算Y轴范围，适配实际数据
const prices = items.flatMap(item => [
  item.shfe_price_cny_per_g, 
  item.london_price_cny_per_g
]).filter(v => v != null);

let yMin = 800;
let yMax = 1200;

if (prices.length > 0 && !state.sgeYAxisRange) {
  // 根据数据范围自动调整
  const minPrice = Math.min(...prices);
  const maxPrice = Math.max(...prices);
  const range = maxPrice - minPrice;
  const padding = Math.max(5, range * 0.2);  // 至少5元边距
  
  yMin = Math.floor(minPrice - padding);
  yMax = Math.ceil(maxPrice + padding);
} else if (state.sgeYAxisRange) {
  // 用户手动缩放后使用用户设置
  yMin = state.sgeYAxisRange.min;
  yMax = state.sgeYAxisRange.max;
}
```

## 技术细节

### Y轴计算逻辑

```javascript
// 1. 提取所有价格数据
const prices = items.flatMap(item => [
  item.shfe_price_cny_per_g,      // 人民币金价
  item.london_price_cny_per_g     // 国际金折算
]).filter(v => v != null);

// 2. 计算范围
const minPrice = Math.min(...prices);  // 最小值
const maxPrice = Math.max(...prices);  // 最大值
const range = maxPrice - minPrice;     // 数据范围

// 3. 添加边距（20%或至少5元）
const padding = Math.max(5, range * 0.2);

// 4. 计算Y轴
yMin = Math.floor(minPrice - padding);  // 向下取整
yMax = Math.ceil(maxPrice + padding);   // 向上取整
```

### 示例计算

**当前数据**：
```
prices = [1002.71, 998.81]
minPrice = 998.81
maxPrice = 1002.71
range = 3.9

padding = max(5, 3.9 * 0.2) = max(5, 0.78) = 5

yMin = floor(998.81 - 5) = 993
yMax = ceil(1002.71 + 5) = 1008

最终Y轴范围：993 ~ 1008 (跨度15元)
```

**放大倍数**：
```
修改前：400元跨度 → 3.9元差异占 0.975%
修改后：15元跨度  → 3.9元差异占 26%

放大：26% / 0.975% ≈ 27倍
```
