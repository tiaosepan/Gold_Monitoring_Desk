## 🖥️ 前端筛选逻辑

### 代码位置：`static/app.js`

#### 筛选函数（第970-973行）

```javascript
function filterEvents(events) {
  if (state.eventFilter === "all") return events;
  return events.filter((item) => item.event_type === state.eventFilter);
}
```

## 💡 可能的原因

### 原因1: 原系统的战争进度事件较少

原系统可能：
- 战争进度事件占比很小
- 或者最新的50条中没有战争进度事件
- 所以筛选后显示为空

### 原因2: 分类标准不同

可能原系统和我们的分类标准不同：
- 我们认为是"war"的事件
- 原系统可能分类为"political"或其他

### 原因3: 这是正常现象

如果最近的RSS源主要发布政治缓和类新闻，那么：
- "全部"模式：显示所有事件
- "战争进度"模式：筛选后可能为空（因为最近没有战争类新闻）
