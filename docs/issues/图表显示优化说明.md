# 图表显示优化说明

**优化时间**: 2026-03-28  
**优化范围**: SGE 溢价走势、黄金反转监控、美债联动图表

## 问题描述

### 1. 配色方案不一致
**原问题**: 复刻系统使用砖红/青绿/金色配色，与原系统的蓝/绿/橙黄不一致

**影响**: 视觉风格不符合原系统

### 2. X 轴时间显示混乱
**原问题**: X 轴显示出现 "20:00, 21:00, 22:00, 23:00, 28, 01:00..."，日期"28"混在时间标签中

**影响**: 时间轴阅读困难

### 3. Y 轴溢价范围不合理
**原问题**: 右 Y 轴（溢价）显示 0～3.5，起点从 0 开始不合理

**影响**: 图表纵向空间利用不佳

### ✅ 2. X 轴时间格式优化

**修复逻辑**:
```javascript
xAxis: { 
  type: "time",
  axisLabel: { 
    formatter: function(value) {
      const date = new Date(value);
      const hours = date.getHours();
      const minutes = date.getMinutes();
      
      // 零点时显示日期（MM-DD格式）
      if (hours === 0 && minutes === 0) {
        return `${String(date.getMonth() + 1).padStart(2, '0')}-${String(date.getDate()).padStart(2, '0')}`;
      }
      
      // 其他时间只显示时:分
      return `${String(hours).padStart(2, '0')}:${String(minutes).padStart(2, '0')}`;
    }
  },
  // 添加竖向虚线分隔
  splitLine: { show: true, lineStyle: { color: "rgba(18,32,51,0.05)", type: "dashed" } }
}
```

**修复效果**:
- ❌ 修改前: `20:00, 21:00, 22:00, 23:00, 28, 01:00, 02:00...`（混乱）
- ✅ 修改后: `20:00, 21:00, 22:00, 23:00, 03-28, 01:00, 02:00...`（清晰）
