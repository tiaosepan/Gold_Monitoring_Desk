### 4.2 数据采集实现

**新浪财经API调用**：
```python
import aiohttp

async def fetch_sina_data():
    """
    采集新浪财经数据
    返回格式：var hq_str_XXX="字段1,字段2,..."
    """
    codes = [
        "gds_AU9999",  # 沪金99
        "hf_XAU",      # 伦敦金现货
        "USDCNY"       # 美元人民币
    ]
    
    url = f"https://hq.sinajs.cn/list={','.join(codes)}"
    
    async with aiohttp.ClientSession() as session:
        async with session.get(url) as response:
            text = await response.text()
            return parse_sina_response(text)
```

**数据解析**：
从 `raw_payload` 可以看出新浪数据格式：
```
var hq_str_gds_AU9999="1022.50,0,1022.50,1023.30,1028.99,965.10,10:48:12,..."
```

解析逻辑：
- 沪金99（gds_AU9999）：第1个字段是价格（元/克）
- 伦敦金（hf_XAU）：第1个字段是价格（美元/盎司）
- 汇率（USDCNY）：第2个字段是汇率

## 十三、关键发现

### 13.1 数据采集机制
- **轮询频率**：60秒（SGE和美债）
- **RSS频率**：1800秒（30分钟）
- **数据源**：完全依赖新浪财经API（免费）

### 13.2 市场判断逻辑
系统会检查：
- `both_markets_open`：两个市场是否同时开盘
- `shfe_market_open`：上海市场是否开盘
- `london_market_open`：伦敦市场是否开盘

未同时开盘时跳过告警。

### 13.3 溢价计算公式
```python
# 国际金价转换为人民币/克
london_price_cny_per_g = (london_price_usd_per_oz * usdcny_rate) / 31.1035

# 计算溢价
premium_cny_per_g = shfe_price_cny_per_g - london_price_cny_per_g
```
