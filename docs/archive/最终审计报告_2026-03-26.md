#### 2. 事件分类逻辑不准确 [MEDIUM]
**问题**: 简单地优先返回war，导致"peace talks"被错误分类  
**影响**: 事件分类不符合原系统逻辑  
**修复**: 
```python
# 文件: backend/services/rss_collector.py (第107-166行)
# 修改前: 简单优先war
# 修改后: 分层判断

def classify_event(self, title, summary, source_category):
    content = f"{title} {summary}".lower()
    
    # 优先级1: 谈判类关键词 → political
    negotiation_keywords = ['talks', 'negotiation', 'ceasefire', ...]
    for kw in negotiation_keywords:
        if kw in content:
            return 'political'
    
    # 优先级2: 纯战争类关键词 → war
    pure_war_keywords = ['attack', 'bombing', 'military', ...]
    for kw in pure_war_keywords:
        if kw in content:
            return 'war'
    
    # 优先级3: 战争恢复类关键词 → war
    war_recovery_keywords = ['resum', 'reopen', 'restore', ...]
    for kw in war_recovery_keywords:
        if kw in content:
            return 'war'
    
    # 优先级4: 其他政治关键词 → political
    # 优先级5: 使用源分类
```
