# 快速参考卡片 - v1.1.0

## 🚀 启动系统

```bash
.\start.bat
```

## 📋 常用命令

### 测试相关
```bash
# 运行所有测试
pytest tests/ -v

# 带覆盖率
pytest tests/ --cov=backend/services --cov-report=html

# 查看覆盖率报告（浏览器）
start htmlcov\index.html
```

### 数据源检查
```bash
# 检查所有数据源
python scripts\utils\check_data_sources.py

# 禁用异常RSS源
python scripts\utils\disable_bad_rss_sources.py
```

### 性能测试
```bash
# 测试执行时间
python scripts\utils\test_scheduler.py --mode timing

# 长时间监控（5分钟）
python scripts\utils\test_scheduler.py --mode monitor --duration 300
```

### 数据库维护
```bash
# 添加索引
python scripts\utils\add_database_indexes.py

# 重建数据库
python recreate_db.py
```

## 📁 日志文件

| 日志文件 | 说明 |
|---------|------|
| `logs/main.log` | 主程序日志 |
| `logs/scheduler.log` | 调度器日志 |
| `logs/sge_monitor.log` | SGE监控日志 |
| `logs/*_error.log` | 各模块错误日志 |

### 查看日志
```powershell
# 实时监控
Get-Content logs\scheduler.log -Wait -Tail 20

# 查看错误
Get-Content logs\*_error.log -Tail 50
```

## 🔧 配置FRED API（推荐）

```powershell
# 获取密钥: https://fred.stlouisfed.org/docs/api/api_key.html

# 临时设置
$env:FRED_API_KEY="your_key_here"

# 永久设置
[System.Environment]::SetEnvironmentVariable("FRED_API_KEY", "your_key_here", "User")
```
