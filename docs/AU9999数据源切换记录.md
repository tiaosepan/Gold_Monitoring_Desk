## 修改目的
将人民币金价数据源从 `nf_AU0`（沪金连续期货）切换到 `gds_AU9999`（上海金现货），与参考网站 http://rev.sccit.com.cn:8000/ 保持数据一致。

## 修改详情

### 1. 新浪财经 API 数据代码映射（`backend/utils/sina_api.py`）

**修改前**：
```python
SYMBOLS = {
    'sge_au9999': 'nf_AU0',          # 沪金连续（替代gds_AU9999以获取实时数据）
    ...
}
```

**修改后**：
```python
SYMBOLS = {
    'sge_au9999': 'gds_AU9999',      # AU9999上海金现货（替代nf_AU0期货以获取现货数据）
    ...
}
```

### 2. 数据解析逻辑更新（`backend/utils/sina_api.py`）

**修改前**（期货数据格式）：
- 字段5 = 最新价（期货）
- 字段6 = 买价/当前价
- 字段8 = 昨结算价

**修改后**（现货数据格式）：
- 字段2 = 现价（现货）
- 字段3 = 开盘价
- 字段6 = 昨收价

```python
# gds_AU9999格式(上海金现货): 日期,时间,现价,开盘,最高,最低,昨收,涨跌,涨跌幅,...
# 字段0=日期, 字段1=时间, 字段2=现价(最重要)
# 字段3=开盘, 字段4=最高, 字段5=最低, 字段6=昨收
# 优先顺序: 字段2(现价) -> 字段3(开盘) -> 字段6(昨收)
```
